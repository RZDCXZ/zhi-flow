# 14 独立检索实验页

Type: task
Status: ready-for-agent
Blocked by: 13

## 学习目标

将检索质量与答案生成分离，直观看到查询向量、Top-K、过滤、分数和来源元数据。

## 前置依赖

- 里程碑 13 已完成并获准继续。

## 最小实现

- 建立仅用于学习调试的 Retrieval Lab，可选 Knowledge Base、输入查询并召回默认 20 个 Chunk。
- 展示排名、距离/归一化分数、Document、页码、标题路径、原文和查询耗时。
- 允许在安全范围调整 Top-K，不允许从浏览器改变模型、密钥或原始 SQL。

## 运行观察

- 使用同义词、专有名词、无关问题和跨文档问题，观察召回成功与失败。
- 比较 Top-K 对召回和噪声的影响。

## 验证方法

- HTTP 接缝验证输入、Knowledge Base 隔离、排序、上限和错误。
- 浏览器冒烟验证候选与来源能被检查。

## 完成标准

- 无需调用聊天模型即可重现和解释一次检索；页面不暴露敏感配置。
- 展示典型成功/失败查询，等待用户决定是否进入里程碑 15。

## 本阶段暂不处理

- Reranker、问题改写、答案、Citation 和评测汇总。
